## [0.1.0] - 2026-06-12

### Added

- Initial `codescent` Python package, Typer CLI, FastMCP stdio server, MIT
  license, and baseline local documentation.
- Python-first MVP for local, source-read-only indexing, bounded search,
  deterministic code-health findings, finding context, refactor planning,
  suggested verification commands, finding lifecycle, and rescan.
- SQLite-backed `.codescent/` state with migrations, locking, project config,
  scan history, and finding state.
- Search expansion for bounded multi-search, changed files, TODOs, tests,
  pagination, ranking, and frecency.
- Graph and context tools for references, callers, callees, related files, and
  impact analysis.
- Reporting and finding workflow surfaces for detail, score explanation,
  backlog, progress, regressions, JSON export, and Markdown export.
- Recommend-only verification and diff-risk workflows, including
  `verify_change`, `review_diff_risk`, and `get_changed_file_health`.
- Configuration, rules, watch, safe reset, workflow prompt resources, and
  optional agent-routing templates.
- Python and TypeScript/React/Next language and rule packs.
- Local CI/PR review commands and opt-in subjective review with runtime network
  disabled unless explicitly enabled.
- loopback dashboard for health, findings, progress, rule config updates, and
  exports.
- Deterministic evals, agent-in-the-loop eval guidance, real-repo smoke, MCP
  smoke, dashboard smoke, and source-read-only safety proofs.

### Fixed

- Preserved project metadata while updating rule config.
- Rendered root config keys before TOML tables.
- Closed final PRD-remainder QA issues found during runtime safety, dashboard,
  MCP/CLI, and config verification passes.

### Docs

- Added PRD, architecture, MCP surface, eval, language-pack, and agent-routing
  docs for the shipped local surface.

### Tests

- Added unit, integration, contract, smoke, eval, docs, and security coverage
  for the shipped CLI, MCP, dashboard, storage, language-pack, and safety
  surfaces.
